Home

SaveMyGeo is a small application that uses the geolocation as a key driver to interact. It has just 2 functions are handled to perfection that are:

- Localizate and save your location: This function does is capture your current location on the map provided by Google Maps and save to locally on your device or computer.

- Load a saved location: This function is intended load a location that you find previously saved, so that you can see Map denuevo in more detail or activating the option "Watch streets."

Enjoy using the application! :D
